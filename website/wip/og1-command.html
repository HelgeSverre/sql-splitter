Command

split

Split a SQL dump file into individual table files at 600+ MB/s

$ sql-splitter split <INPUT> [OPTIONS]

Key Options

--output, -o Output directory
--dialect mysql | postgres | sqlite
--tables Filter specific tables
--schema-only DDL statements only
; sql-splitter

Example Output

# Split database.sql into tables/
$ sql-splitter split database.sql -o tables/

# Output structure:
tables/
├── users.sql
├── orders.sql
└── _global.sql
